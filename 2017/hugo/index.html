Hugo搭建个人博客到GitHub Pages

Written by PengSite with ♥ on September 28, 2017 in Uncategorized

  1. Windows下安装Hugo以及Git工具

    到Github下载Hugo，下载对应的操作系统版本的Hugo文件。

    到Git网站下载对应的操作系统版本的Git文件。

    下载完成之后解压缩，我直接解压到桌面blog文件夹下进行管理(不建议桌面)。然后添加环境变量；我的路径为:C:\Users\Administrator\Desktop\blog\hugo;也就是把此路径加入环境变量。

    添加环境变量的方法:设置->系统->关于->高级系统设置->环境变量->path->新建->确定

    检查一下是否安装成功：

    cmd键入命令:
hugo version
#出现相应版本号即为安装成功
# hugo v0.82.1-60618210 windows/amd64 BuildDate=2021-04-20T11:02:50Z VendorInfo=gohugoio

  2. 用Hugo创建自己的博客

    在hugo同名文件下使用Git命令快速生成站点:

    # 您也可以 hugo new site /path/to/site  
# 比如希望生成到 /path/to/site 路径
hugo new site myblog

    这样就在 myblog 目录里生成了初始站点，cd myblog进入目录 ls 查看目录结构 ：

      ▸ archetypes/
  ▸ content/
  ▸ layouts/
  ▸ static/
  ▸ themes/
    config.toml

    其中 content 文件夹存放博文，layouts 文件夹存放博客模版，比如博客的页面结构。static 文件夹存放前端文件和以后显示在博客中的图片，themes 存放主题文件，安装的主题都会存放在该文件夹中。config.toml 是博客的配置文件。

  3. 选择主题并下载安装

    到hugo官网下载主题，下面是我比较喜欢的博客主题，比较简约:

    https://github.com/liuzc/leaveit

    您可以直接下载到themes/文件夹下并解压，然后进入LeaveIt/exampleSite文件夹，复制content和config.toml文件到myblog下覆盖掉原本的content和config.toml；

    之后到本地上跑一下把

    hugo server --buildDrafts -w

    输入http://localhost:1313/看一下吧

    输入http://localhost:1313/可以看到页面啦，至此hugo本地搭建就已经完成了。

    接下来比较重要 : 回到Git->Ctrl+C停止hugo运行,并键入命令hugo;在myblog目录下就可以看到一个public文件，为下面同步到Github做准备。

  4. 同步到 Github

    1. 在Github上创建一个仓库，仓库的名字是你的名字.github.io之后点击Create repository创建完成。

    2. 接下来把上述生成的public静态页同步到Github

      进入到public文件夹内，用Git命令终端键入一下命令(第一次使用Git需填入Github信息):

      echo "# 你的名字.github.io" >> README.md
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/你的地址/你的地址.github.io.git
git push -u origin main

      之后就完成了同步，如果在push过程中出现问题可以看下相关Git操作。

    3. 查看自己博客的地址

      点击Settings->Pages就可以看到自己的博客地址啦。

  5. 到此Hugo博客部署到Github就结束了。
