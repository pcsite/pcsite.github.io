枚举

  • 是将变量的值一一列出来，变量的值只限于列举出来的值得范围内

  • 为了间接的表示一些固定的值，java就给我们提供了枚举

  • 枚举的格式

  • 定义枚举类需要用到关键字enum

    public enum s{
	枚举1，枚举2，枚举3;
}

  • 练习

  • 定义一个枚举类，用来表示春、夏、春、冬这四个固定值

    public enum Season {
    SPRING,SUMMER,AUTUMN,WINTER;
}

  • 枚举的特点

    • 所有枚举类都是Enum的子类

    • 我们可以通过"枚举类名.枚举项名称"去访问指定的枚举项

    • 每一个枚举项其实就是该枚举的一个对象

    • 枚举也是一个类，也可以去定义成员变量

    • 枚举类的第一行上必须是枚举项，最后一个枚举项后的分号是可以省略的，但是如果枚举类有其他的东西，这个分号就不能省略，建议不要省略

    • 枚举类可以有构造器，但是必须是private的，他默认也是private的，枚举项的用法比较特殊：枚举("");

    • 枚举类也可以有抽象方法，但是枚举项必须重写该方法

    }
}

  • 枚举常见的方法

    方法名 说明
    String name() 获取枚举项的名称
    int ordinal() 返回枚举项在枚举类中的索引值
    int compareTo(E o) 比较两个枚举项，返回的是索引值的差值
    String toString() 返回枚举类常量的名称
    static TvalueOf(Class type,String name) 获取指定枚举类中的指定名称的枚举值
    values() 获得所有的枚举项

    • 测试类

      public class EnumDemo {
    public static void main(String[] args) {
//        String name()获取枚举项的名称
        String name = Season.SPRING.name();
        System.out.println(name);
        System.out.println("----------------");
//        int ordinal()返回枚举项在枚举类中的索引值
        int ordinal = Season.AUTUMN.ordinal();
        System.out.println(ordinal);
//        int compareTo(E o)比较两个枚举项，返回的是索引值的差值
        int i = Season.SPRING.compareTo(Season.WINTER);
        System.out.println(i);
//        String toString()返回枚举类常量的名称
        String s = Season.SPRING.toString();
        System.out.println(s);
//        static<T> T valueOf(Class<T> type,String name)获取指定枚举类中的指定名称的枚举值
        Season season = Enum.valueOf(Season.class, "SPRING");
        System.out.println(season);
        System.out.println(Season.SPRING == season);
//        values()获得所有的枚举项
        Season[] values = Season.values();
        for (Season value : values) {
            System.out.println(value);
        }
    }
}

  • 注解

    • 什么是注解

      • 对我们的程序进行标注和解释
        注解名 说明
        @Override 描述子类重写父类的方法
        @Deprecated 描述方法过时
        @SuppressWarnings 压制警告

    • 自定义注解

      • 格式

        public @interface 注解名称{
	public 属性类型 属性名 () default 默认值;
}

      • 例

        public @interface Test{
public int a () default 23;
}

      • 代码演示

        public @interface Anno1 {
      
    //定义一个基本类型的属性
    int a () default 23;
      
    //定义一个String类型的属性
    public String name();
      
    //定义一个Class类型的属性
    public Class clazz() default Anno2.class;
      
    //定义一个注解类型的属性
    public Anno2 anno2() default @Anno2;
      
    //定义一个枚举类型的属性
    public Season season() default Season.SPRING;
      
    //以上类型的一维数组
    //int数组
    public int[] arr() default {1,2,3,4,5};
      
    //枚举数组
    public Season[] season1() default {Season.SPRING,Season.SUMMER};
}

      • 特殊属性value

        • 后期我们在使用注解的时候，如果我们只需要给注解的value的属性赋值，那么value就可以省略
        • public String value();

    • 自定义注解练习

      • 自定义一个注解(@Test)，用于指定类的方法上，如果某一个类的方法上使用了该注解，就执行该方法。

      • isAnnotationPresent(Class<? extends Annotation> annotationClass)

        • 判断当前方法上是否有指定的注解
        • 参数：注解的字节码文件对象
        • 返回值是boolean

      • 测试类

        public class AnnoDemo {
    public static void main(String[] args) throws ClassNotFoundException, InstantiationException, IllegalAccessException, InvocationTargetException {
        //1.通过反射获取UseTest类的字节码文件对象
        Class aClass = Class.forName("com.itheima.myanno1.UseTest");
        //创建对象
        UseTest useTest = (UseTest) aClass.newInstance();
      
        //2.通过反射获取这个类里面所有的方法对象
        Method[] declaredMethods = aClass.getDeclaredMethods();
        //遍历数组
        for (Method declaredMethod : declaredMethods) {
            //declaredMethod表示每一个方法对象
            //isAnnotationPresent(Class<? extends Annotation> annotationClass)
            //判断当前方法上是否有指定的注解
            //参数：注解的字节码文件对象
            //返回值是boolean
            if (declaredMethod.isAnnotationPresent(Test.class)){
                declaredMethod.invoke(useTest);
            }
        }
    }
}
